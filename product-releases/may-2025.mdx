You told us you wanted a faster and more organized way to manage your projects, datasets, and API keys. We listened. Today, we are excited to announce a completely redesigned workflow experience.

Our May 2025 update introduces a new landing page, a dedicated datasets section, and a secure place to manage your API keys and much more. These changes will help you work faster, stay organized, and get more done.

---

## Your New Landing Page

We redesigned the welcome experience to help you get started faster. The new landing page is your central hub for everything you need.

![Landing Page](https://storage.googleapis.com/labellerr-cdn/%201%20%20Product%20Update%20May%202025/landing-page.webp)

**What's new:**
- A quick video guide helps you learn the basics right away
- **Quick Actions**: Start working in seconds with one-click access to:
  - **Recently Opened**: Jump back into your most recent project
  - **Create Project**: Start a new annotation project for your data
  - **Create Dataset**: Create a new dataset without creating a new project
  - **New API Key**: Generate a new API key for the Labellerr SDK
  - **Explore Your Data**: Run smart queries using Labellerr AI on your datasets

![Side Panel](https://storage.googleapis.com/labellerr-cdn/%201%20%20Product%20Update%20May%202025/side-panel.webp)

**Easy Navigation**: The new sidebar helps you find what you need:
- Home
- Projects
- Datasets
- API Keys
