## Polyline Annotations (NEW)

![Polyline Annotation](https://storage.googleapis.com/labellerr-cdn/%201%20%20Product%20Update%20May%202025/polyline.webp)

We're excited to introduce polyline annotations - a new annotation type that expands your labeling capabilities.

**Versatile Use Cases:**

Perfect for annotating linear features like:
- Road lanes and traffic paths
- Power lines and cables
- Boundary lines and fences
- Sequential workflows or processes

**Enhanced Functionality:**
- Add custom attributes directly to polyline annotations for richer data capture
- Create smooth curves for more natural and precise representations
- Maintain full compatibility with existing annotation workflows

## Annotated File Similarity

![File Similarity](https://storage.googleapis.com/labellerr-cdn/%201%20%20Product%20Update%20May%202025/file-similarity.webp)

**Key Features:**
- **Intelligent Duplicate Detection**: Click the magnifying glass icon in the right panel of the labeling screen to check if your current file is similar to previously annotated files
- **Adjustable Similarity Threshold**: Fine-tune the similarity index from loose similarity detection to precise duplicate identification
- **Workflow Optimization**: Reduce redundant annotation work by leveraging existing labels from similar files

## Dataset Management Improvements

Flexible Dataset Operations: We've streamlined dataset deletion and detachment with intelligent workflow awareness:

- **Direct Deletion**: Delete datasets that aren't attached to any projects instantly
- **Smart Detachment**: For datasets attached to projects but with unlabeled files, easily detach and delete through the improved interface
- **Work-in-Progress Handling**: For datasets in progress or review stages, reassign files to "unlabeled" status via the three-dot menu, then detach and delete as needed
