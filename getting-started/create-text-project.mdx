## Create a New Text Project

<Steps>
  <Step title="Selecting Data">
    Once at the **New project** page, To create a new project, first select an existing project name at the top. A popup will open displaying all projects within the workspace, along with a 'Create New Project' button. Click on 'Create New Project' to begin.

    Give your project a suitable name and select the **Text** data type. Then click on **Save and Next**.

    <Warning>
    Only one data type can be selected per project
    </Warning>

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/f6ecf91b-399a-4975-b3fe-c209c00e0991.webp" alt="Create text project screen" />
    </Frame>
  </Step>

  <Step title="Upload/Attach existing dataset">
    If a new dataset is to be created, Click on the "Create Dataset" button. The available formats are Google Drive, Google Cloud, Amazon S3 & Local Upload.

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/bbcd4c02-6600-47e8-a440-c476b3678c00.webp" alt="Text dataset upload options" />
    </Frame>

    <CardGroup cols={2}>
      <Card>
        <Frame>
          <img src="https://cdn.labellerr.com/1%20%20Documentation/cee24db1-c08d-4102-8cbf-17eff2c874bb.webp" alt="Upload option 1" />
        </Frame>
      </Card>
      <Card>
        <Frame>
          <img src="https://cdn.labellerr.com/1%20%20Documentation/a7be3923-ef84-47f5-9b11-19c0b421e958.webp" alt="Upload option 2" />
        </Frame>
      </Card>
    </CardGroup>

    Next, fill in the new dataset details: Enter the Name (mandatory) and Description (optional), then click on 'Create Dataset'.

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/f76dd5cd-812f-4df0-ab63-c013421c244b.webp" alt="Text dataset details form" />
    </Frame>
  </Step>

  <Step title="Label Configuration">
    Here, Objects and Classifications based on the labeling requirements are supposed to be created. You can create a new template from scratch using the various options available or select an existing template from previous work on clicking **'Existing Templates'** button.

    There are different 'Object' & 'Classification' types. For objects, in **Objects** tab, Click on '**Add Object**'. Enter Object name and select object type. Click on ðŸ’¾ button to add it.

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/289edd99-a881-47fb-8280-c7efa23b7a62.webp" alt="Text label configuration" />
    </Frame>

    To add attributes of an object or to edit the type/name of object, click on pen icon in parallel to object name. Click on 'Add Attribute'. Enter name of the Attribute and click on 'Add Options' to enter the values of attribute.

    <Info>
    **Additional options are also available:**
    - **Required** - Indicates that the attribute must be completed
    - **Sort options** - Allows sorting the values of the attribute  
    - **Not sure remarks** - Enables adding remarks when unsure about an answer
    </Info>

    <CardGroup cols={2}>
      <Card>
        <Frame>
          <img src="https://cdn.labellerr.com/1%20%20Documentation/3ef03cd7-a9c7-4ebd-a3db-f45d480b47fa.webp" alt="Text attribute interface 1" />
        </Frame>
      </Card>
      <Card>
        <Frame>
          <img src="https://cdn.labellerr.com/1%20%20Documentation/7b374387-672b-4820-bcf2-a46f75d8e694.webp" alt="Text attribute interface 2" />
        </Frame>
      </Card>
    </CardGroup>

    <Tip>
    Similarly, for classification, navigate to the 'Classifications' tab. Click on 'Add Classification', then enter the classification name/question. Select the classification type from the dropdown options, and click on 'Add Attribute' to enter values for dropdown, radio, or multiselect options.
    </Tip>

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/873f6c12-3f75-4d4e-abfc-43e842767591.webp" alt="Text classification interface" />
    </Frame>

    After you've finished, click on the "**Next Step**" button. Then, fill your new "**Template Details**" details and click on "**Proceed**" to continue so you can use the same template in future projects.

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/3cd08c7f-9c94-43e5-9dd8-43d5614e3a36.webp" alt="Text template details" />
    </Frame>
  </Step>

  <Step title="Review">
    Select your review process by enabling or disabling the toggle button, Enable review cycle (if **Review Process** is needed) & Enable client review cycle (if **Client Review Process** is needed).

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/5b5eaaf9-acc7-4a66-8971-424c4207d73a.webp" alt="Text review settings" />
    </Frame>

    <Tip>
    Click the 'Create Project' button to create your project. Once created, it will appear in the list along with your previously created projects. To view a project, simply click the 'View button'
    </Tip>

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/dbc3762d-e713-4c06-8475-ac32262dcd6a.webp" alt="Text project created" />
    </Frame>

    <Info>
    **To find the project ID**, check the URL in the web link of the project's dashboard, labeling, review/client review screens, or files page.
    </Info>

    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/4687ef0e-8787-45f8-968b-3b63a938b819.webp" alt="Text project ID location" />
    </Frame>
  </Step>
</Steps>
