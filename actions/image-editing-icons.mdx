Labellerr provides additional tools to edit image samples for a better labelling experience.

## Available Tools

### Move/Drag Tool
**Move/Drag** icon lets you move the Image after selecting it. This is useful when you want a certain edge of the image or in complex images where detailed annotating is required.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/11485782-75ed-4b3a-9e5a-17bdf4e45d72.webp" alt="Move/Drag Tool" />
</Frame>

### Display Settings
**Display** icon lets you change the brightness, contrast, hue, saturation and luminance. It is especially useful for details that are often missed in a very dark/bright picture or for the detection of an object that blends with the environment.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/1cccb310-5214-476a-8eb8-3db89210c839.webp" alt="Display Settings" />
</Frame>

### Radius Tool
**Radius** icon allows edge reduction or anchor radius editing in different type of annotations like bounding box/polygon/line/segmentation.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/4bb8ef1d-f740-44d6-9a65-88dc57ce8a28.webp" alt="Radius Tool" />
</Frame>

### Zoom Tool
**Zoom** icon as the name suggests lets you zoom in and zoom out. This can be useful when the image pixels are huge and specific pixels have to be highlighted with precision. It also serves as useful while using dot annotation in high pixel images.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/d9c01718-6330-416c-b5e7-cdd3d4d9caa0.webp" alt="Zoom Tool" />
</Frame>

### Reset Tool
**Reset** icon allows you to reset the file to its original position.

  <img src="https://cdn.labellerr.com/1%20%20Documentation/e3e9596b-53c1-4642-8b53-30d610fff080.webp" alt="Reset Tool" />
