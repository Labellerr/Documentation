### Grid View

After applying the filters, the grid view can be enabled by clicking on the grid icon in the top right section of the page.

This view enables the users to easily see the annotation present within the filtered files at a glance.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20docupdate/gridexport.webp" alt="Grid View" />
</Frame>

### Object View

With object view, you can access each annotated object individually and can change the 'label' name to correct one while reviewing.
