### Statistics View

To see the stat view with histogram analysis to display the frequency of labels, showing the number of occurrences for each label name and the status of the files.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/7db5b1b6-54ad-4997-be57-904af948d3a5.webp" alt="Statistics View" />

### Grid View

After applying the filters, the grid view can be enabled by clicking on the grid icon in the top right section of the page.

This view enables the users to easily see the annotation present within the filtered files at a glance.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/7dd25ae1-d952-4448-ab24-d874f37bef31.webp" alt="Grid View" />
</Frame>

### Object View

With object view, you can access each annotated object individually and can change the 'label' name to correct one while reviewing.
