### Key Elements and Functionalities

*   On the left there is **objects and classification** questions that were created while making a new project.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20labellerr_productupdate/bowler%20detected.webp" alt="Tools and Classifications" />
</Frame>

*   **Mandatory questions** must be answered before submission but others can be skipped if not needed.

*   To use the **Autolabel** feature, an auto label model first needs to be attached.

*   ‘**Clear all answers’** removes the previous entries for a sample if corrections need to be made. Alternatively, you can also delete a single annotation using the delete button against that annotation.

*   If the classifications list is long, then a user can also search for the desired question pertaining to the current sample data from the **Search Question** option.

*   The **Settings option** can be used to update/change the annotation questions for the project.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20labellerr_productupdate/Change%20annotation.webp" alt="Settings Option" />
</Frame>

*   If the sample is too distorted, blurry, or irrelevant then, the user can instead **skip** the sample and add a remark stating the reason for skipping.

<Frame>
  <img src="https://cdn.labellerr.com/1%20%20Documentation/99dd7e70-2d0d-41ef-8254-d015e5ecfa59.webp" alt="Skip Option" />
</Frame>

*   After each sample is done being marked, click on **SUBMIT** present in the bottom left corner beneath the objects and classifications.

*   On the top center, the tool also shows how much time has been spent labelling the particular set and beneath it, the time spent at each session.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20labellerr_productupdate/tools_time.webp" alt="Time Spent Tracking" />
</Frame>

*   To close the current session, simply press the HOME button on the top left corner.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20labellerr_productupdate/home.webp" alt="Exit Button" />
</Frame>

*   You also have the Move image, Display image, Radius, Zoom In-Out, Reset icons, selection Tool available on top of each Image file to set the image quality, display and position.

<Frame>
  <img src="https://storage.googleapis.com/labellerr-cdn/%20%20%201%20labellerr_productupdate/tools_time.webp" alt="Image Editing Tools" />
</Frame>
