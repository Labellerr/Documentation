An annotation can also be edited based on classification, **Deleted** and **Hidden** easily.

<Info>
**Delete** can be used instead of **clear all answers** option if a single annotation is to be removed.
</Info>

<Info>
Hidden is used when there is overlapping of different annotations. To avoid mistakes, some annotations can be hidden.
</Info>
