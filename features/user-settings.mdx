### Managing Users

<Info>
User settings allows you to add or delete new users and assign new roles and projects to existing annotator
</Info>

<Steps>
  <Step title="Access User Settings">
    To access the user settings, click on the user icon from the settings list on the left.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/00e5bd93-ce95-4a22-9519-ff788ff7cf57.webp" alt="User Icon in Settings" />
    </Frame>
  </Step>

  <Step title="Click on Users Option">
    Click on Users Option.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/c804c3a8-ae06-42f7-9f50-05880d3e6758.webp" alt="Users Option" />
    </Frame>
  </Step>

  <Step title="User Settings Screen">
    The User settings screen should appear which lets you manage users. The following screen shows the list of all users working on a domain. Display on-screen also shows the role and the last date of work for each user. Multiple actions can be taken henceforth.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/d8a77c89-664a-4414-a6b7-c0a0ce66a3b4.webp" alt="User Settings Screen" />
    </Frame>
  </Step>

  <Step title="Search and Edit Users">
    The **search option** allows you to search for a user by name. The **Edit option** lets you edit user credentials, profile and assign or delete roles.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/e3ce3d52-8e83-4eff-a8f0-fc88eedb5e5b.webp" alt="Search and Edit Options" />
    </Frame>
  </Step>

  <Step title="Add a New User">
    To add a new user, simply click on the create user option located on the top right corner.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/0b27e42b-4576-451f-bf79-8988b08b2157.webp" alt="Create User Option" />
    </Frame>
    The following create user screen would appear.
    <Frame>
      <img src="https://cdn.labellerr.com/1%20%20Documentation/6d3f1495-2daf-4a0a-8945-eb54436bec9b.webp" alt="Create User Screen" />
    </Frame>
    <ol>
      <li>On the left, basic user credentials are to be added. Email, First Name and Last name are mandatory fields.</li>
      <li>On the right, click on ‘+’ icon and then projects and roles can be assigned.</li>
    </ol>
  </Step>
</Steps>

Once the input has been made, the new user profile would be made by clicking on save at the bottom right.
